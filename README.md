# GAURUS Project
GNU, Are U Really UrSelf? Re-implementation of GNU Coreutils.

Y'know, I always thought the GNU Core Utilities were simple and minimalist, until i've read its source code; for a simple program like 'true', it has 80 lines of code, and the majority of it
does not need to be written in order to make the program functional. And i'm not only talking about 'true', i'm talking about a lot of programs. I wanted a simple, minimalist way to enjoy using
the GNU Coreutils, and that's why I invented GAURUS (which stands for GNU, Are U Really UrSelf?) which is a re-implementation of the GNU Coreutils, mostly providing the same functionality. I'm still
working on it, and there's A LOT to work on, but I will do things slowly :).

DISCLAIMER: For safety reasons, I do NOT encourage you to fully replace the GNU Coreutils with this project, because it's never guaranteed it will provide the same functionality. If you mindlessly replace all
programs with mine, your system MIGHT break.

This program is distributed in the hope that it will be useful but WITHOUT ANY WARRANTY. The software is provided "as is".
